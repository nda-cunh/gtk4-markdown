# Markdown Gtk4
---------------

This is a simple markdown viewer written in Vala with Gtk4.

## Features

### Basic Syntax 
- [x] Headings
- [x] Paragraphs
- [x] Emphasis (italics, bold, strikethrough, underline)
- [ ] Blockquotes
- [x] Lists
- [x] Code
- [x] Code blocks (no color highlighting)
- [ ] Links
- [x] Images
- [x] Horizontal rules

### Extended Syntax
- [x] Tables
- [ ] Task Lists
- [x] Highlight
- [x] Subscript
- [x] Superscript
- [ ] Automatic URL Linking
- [ ] Footnotes
- [ ] Heading IDs
- [ ] Linking to Heading IDs
- [ ] Definition Lists
- [ ] Strikethrough
- [ ] Emoji
