# Markdown Gtk4
---------------

This is a simple markdown viewer written in Vala with Gtk4.

## Features


### Basic Syntax 
- [x] Headings
- [x] Paragraphs
- [x] Emphasis
- [x] Blockquotes
- [ ] Lists
- [x] Code
- [x] Code blocks (no color highlighting)
- [x] Links
- [x] Images
- [x] Horizontal rules

### Extended Syntax
- [x] Tables
- [x] Task Lists
- [ ] Footnotes
- [ ] Highlight
- [ ] Heading IDs
- [ ] Linking to Heading IDs
- [ ] Definition Lists
- [ ] Strikethrough
- [ ] Subscript
- [ ] Superscript
- [ ] Automatic URL Linking
- [ ] Emoji
