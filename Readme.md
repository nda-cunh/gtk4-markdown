# Markdown Gtk4
---------------

This is a simple markdown viewer written in Vala with Gtk4.

## Features

### Basic Syntax 
- [x] Headings
- [x] Paragraphs
- [x] Emphasis (italics, bold, strikethrough, underline, highlight)
- [x] Blockquotes
- [ ] Lists
- [x] Code
- [x] Code blocks (no color highlighting)
- [x] Links
- [x] Images
- [x] Horizontal rules

### Extended Syntax
- [x] Tables
- [x] Task Lists
- [x] Highlight
- [x] Subscript
- [x] Superscript
- [X] Automatic URL Linking
- [ ] Footnotes
- [ ] Heading IDs
- [ ] Linking to Heading IDs
- [ ] Definition Lists
- [ ] Strikethrough
- [ ] Emoji
